What can be done with Unimacro grammars

With Unimacro grammars lots of things can be done without extensive programming yourself. Many things can/have to be tuned through configuration files, here called inifiles.

Some of the other features:

  • Use the power of AutoHotkey for actions that are difficult to do within Unimacro
  • Going to and do actions with line numbers in IDE programs and for example Excel. For some applications you can choose to jump to a line number module hundred.
  • Putting all sorts of brackets around selection or dictated text (nice start example too!)
  • Numbers lists as spoken forms
  • Calling the Windows key
  • Printing (really pasting) unicode characters like Greek alpha to your sreen
  • Logging all your utterances
  • Control the state of each grammar
  • Configure actions (eg in grammar lines) and lists (eg in grammar folders)
  • Searching (to be developed further) and example of highlight, implemented in Vocola.
  • Translate grammars to other language versions of NaturallySpeaking, or insert synonyms (currently Dutch and English are already available)
  • Displaying messages in the NatSpeak recognition window (rewritten)
  • Present information about current user, active window and state of Unimacro, NatLink and Dragon
